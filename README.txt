
== 产品概述 ==


UEditor是由百度web前端研发部开发所见即所得富文本web编辑器，具有轻量，

可定制，注重用户体验等特点，开源基于BSD协议，允许自由使用和修改代码。




== 安装说明 ==


1. 下载并安装UEditor模块


2. 进入 Home » Administration » Configuration » Content authoring
   
    A: 先进行 Global Settings 的配置 (建议使用默认配置)
   
    B: 配置其它的Profile，可以通过 Create a new profile 创建新的配置
   
    C: 拖动表格左侧滑块可对Profiles排序，系统将优先使用排在前面的Profile

    D: 设置 image style / defualt style / watermark style 控制前台上传图片后的样式处理

    E: 设置 manage image roles 控制前台用户访问“图片”=》“在线管理”权限


3. 文件上传权限做了安全认证处理，可以到 Home » Administration » People，

    给相关的用户组 Access fileupload 权限


4. 本模块适用于Drupal7.x
